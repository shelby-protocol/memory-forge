# MemoryForge — Shelby 提前体验申请书

## 我们是谁

**MemoryForge** 是首个基于去中心化热存储的 MCP 标准 AI agent 持久记忆引擎。一行命令，无限记忆。

| GitHub   | `shelby-protocol/memory-forge` |
| 周下载量 | 7,393                          |
| 许可证   | MIT                            |

## 解决什么问题

AI agent 每次关闭会话就丢失全部上下文。开发者每天早晨要重新解释技术栈、偏好设置和历史决策。团队知识分散在各个独立会话中，无法共享。现有方案要么是中心化云服务（厂商锁定），要么纯本地存储（无法同步、不可审计）。

**MemoryForge 一次性解决三个痛点**：持久上下文、跨设备同步、链上审计——而且零配置，一行命令搞定。

## 部署方式

```bash
# 免费版——30 秒，零外部依赖
npx memory-forge setup

# Pro 版——Shelby 云同步
SHELBY_API_KEY=xxx npx memory-forge pro

# 跨设备——同样记忆跟随你
# 台式机 → Shelby 云 → 笔记本
# SessionStart / Stop 钩子自动同步
```
